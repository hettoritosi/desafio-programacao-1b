# Para executar a aplicação

 1 - Clone o repositório 
 
 2 - Instale as dependências  
 
      $ bundle install 
      
 3 - Crie o banco e migre o schema 
 
      $ rake db:create
      
      $ rake db:migrate

 Agora rode a aplicação
 
      $ rails server
 
 Obs: Caso queira executar os testes basta executar:
 
       $ bundle exec rspec
